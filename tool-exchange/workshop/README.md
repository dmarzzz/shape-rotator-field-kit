# Tool Exchange Workshop

A 4-session series where cohort members install, run, and critique
each other's tools live. Led by [dmarz](https://github.com/dmarzzz).

## Format

Each session is ~10 people max. Rough shape:

- **5 min** — intros, what everyone brought
- **50 min** — two or three submitter demos, live install + dogfood
- **25 min** — critique and integration notes: which tools pair well,
  what gaps we see, what's worth promoting to the kit
- **10 min** — queue for next session, PR status, housekeeping

Everyone brings their laptop and installs each demoed tool in real
time. If your tool doesn't install cleanly in 10 minutes in front of
peers, that is a useful bug report, not a failure. We'd rather catch
it here.

## Sign up

See [`signups.md`](./signups.md). Add your GitHub handle (and an
optional line about what you'll bring or what you're looking to pick
up) to the session you want to attend, open a PR, dmarz merges.

Seats are approximate and first-PR-merged wins. If a session fills
up dmarz will comment on your PR and suggest another slot.

## What to bring

- Your laptop, charged.
- A tool you actually use, or one you're actively building. Rough is
  fine; polish is not a gate.
- A submission in [`tool-exchange/submissions/`](../submissions/)
  (open a PR before the session you plan to demo at).
- Tolerance for live install chaos.
